JWT Authentication Flow

The JWT-based authentication flow allows secure API access:

1. User Authentication
POST /api/auth/login Content-Type: application/json { "username": "user", "password": "password" }
2. Server Issues JWT
{ "token": "eyJhbGciOiJIUzI1NiIsInR5...", "tokenType": "Bearer", "expiresIn": 86400000, "username": "user", "roles": ["ROLE_USER"] }
3. Access Protected Resources
GET /api/resources/user Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5...
